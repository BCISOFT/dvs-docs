# Administration

Commands used for daily administration of your Docker environment.

- **[changelog](changelog)** - Display the changelog
- **[poweron](poweron)** - Start devspaces services
- **[poweroff](poweroff)** - Stop devspaces services
- **[mysqlconsole](mysqlconsole)** - Root access to mySQL / mariaDB
- **[check](check)** - Check and repair apps, remove orphaned resources
- **[status](status)** - List services and applications with their status
- **[recipes](recipes)** - List installed recipes
