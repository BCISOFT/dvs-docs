# Guides

Step-by-step guides to help you get started with DevSpaces and accomplish common tasks.

## What are guides?

Guides provide detailed, step-by-step instructions for specific tasks and workflows. They are designed to help you:

- Set up your development environment
- Work with different types of projects
- Configure and customize DevSpaces
- Troubleshoot common issues
- Learn best practices

## Getting started

If you're new to DevSpaces, start with the [Getting Started](/docs/getting-started/) section to install and configure DevSpaces on your system.

## Available guides

Guides are organized by topic and use case. Browse the sidebar to find guides relevant to your needs.

## Need help?

If you can't find what you're looking for, check the [FAQ](/docs/faq/) section or refer to the [Command Reference](/docs/control-commands/) for detailed command documentation.
