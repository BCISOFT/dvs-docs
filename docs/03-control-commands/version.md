# dvs version

Display version information.

## Description

The `version` command displays the current version of the dvsctl binary and related components.

## Usage

```bash
dvs version
```

## Output Example

```
dvs version 0.3.24
Repository: pro
Installation: MyInstallation
Organization: My company
User: xxx@xxx.com (Test User)
License: pro (monthly) - expires 2025-11-05T09:00:25.044180+00:00
```
