# Application

## Description

Application management.

## Commands

- **[init](init)** - Initializes an application in the current directory
- **[create](create)** - Creates and installs a new application in the current directory
- **[start](start)** - Starts an application
- **[stop](stop)** - Stops an application
- **[list](list)** - Lists installed applications
- **[rebuild](rebuild)** - Rebuilds a Docker image of an application
- **[destroy](destroy)** - Destroys a Docker image of an application
- **[app-folder](app-folder)** - Returns the path of an application
- **[shell](shell)** - Opens a shell in the application container
- **[dump-config](dump-config)** - Exports an application's configuration
- **[logs](logs)** - Displays application logs
