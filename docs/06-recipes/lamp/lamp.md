# LAMP Recipe

## Description

Set of tools adapted to a standard LAMP application: PHP / Apache / MySQL

## Commands

- **[phpinfo](phpinfo)** - Displays the `phpinfo` content
- **[mysql](mysql)** - Shortcut to the `mysql` command
- **[xdebug](xdebug)** - Enables or disables xdebug
- **[composer](composer)** - Shortcut to the `composer` command
- **[php](php)** - Shortcut to the `php` command
- **[db-import](db-import)** - Import a database backup
- **[db-export](db-export)** - Create a database backup

None
