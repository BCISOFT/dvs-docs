## Description

A tunnel allows you to access your local application from outside via a dynamically generated link. This can be useful for sharing with a client, one or more other developers, or for remote development (remote execution and local viewing).

Available only with subscription.

## Commands

- **[tunnel start](tunnel-start)** - Start a sharing tunnel
- **[tunnel stop](tunnel-stop)** - Stop and remove a sharing tunnel
- **[tunnel list](tunnel-list)** - List existing tunnels
- **[tunnel status](tunnel-status)** - Tunnel information
