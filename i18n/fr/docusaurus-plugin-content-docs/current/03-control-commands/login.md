# dvs login

Gère la connexion au registre docker pour DVS.

## Description

La commande `login` s'authentifie auprès du registre Docker configuré pour votre installation DevSpaces et permet l'accès aux fonctionnalités supplémentaires si votre licence vous le permet.

## Utilisation

```bash
dvs login
```

## Utilisation

```bash
# Se connecter au registre
dvs login
```

## Notes

Utile principalement pour forcer un login si la connexion automatique échoue.
