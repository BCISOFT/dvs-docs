# dvs check-upgrade

Vérifie si une mise à jour est disponible.

## Description

La commande `check-upgrade` consulte la liste des dernières versions disponible pour chaque composant et indique si une mise à jour est nécessaire.

## Utilisation

```bash
dvs check-upgrade
```

## Commandes associées

- `dvs upgrade` - Télécharge et installe la dernière version des composants
