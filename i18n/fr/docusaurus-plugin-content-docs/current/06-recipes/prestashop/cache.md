# cache

Active ou désactive le mode cache de prestashop

## Usage

```bash
dvs cache [action]
```

## Argument

- `action` - commande à choisir parmi:
    - `status`: Affiche le status actif/inactif
    - `toggle`: Bascule le statut
    - `enable`: Active le mode cache de prestashop
    - `disable`: Désactive le mode cache de prestashop
