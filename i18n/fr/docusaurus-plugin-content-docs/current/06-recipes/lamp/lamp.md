# Recette LAMP

## Description

Ensemble d'outil adapté à une application LAMP standard: PHP / Apache / mySQL

## Commandes

- **[phpinfo](phpinfo)** - Affiche le contenu du `phpinfo`
- **[mysql](mysql)** - Raccourci vers la commande `mysql` 
- **[xdebug](xdebug)** - Active ou désactive xdebug
- **[composer](composer)** - Raccourci vers la commande `composer`
- **[php](php)** - Raccourci vers la commande `php`
- **[db-import](db-import)** - Import d'une sauvegarde base de données
- **[db-export](db-export)** - Création d'une sauvegarde base de données
