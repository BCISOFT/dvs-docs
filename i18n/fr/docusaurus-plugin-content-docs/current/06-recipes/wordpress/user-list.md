# user-list

Liste les comptes utilisateur du site.

## Usage

```bash
dvs user-list
```

## Exemple

```bash
$ dvs user-list
+----+------------+--------------+------------+---------------------+---------------+
| ID | user_login | display_name | user_email | user_registered     | roles         |
+----+------------+--------------+------------+---------------------+---------------+
| 1  | wp         | wp           | wp@dvs.sh  | 2025-09-10 10:40:30 | administrator |
+----+------------+--------------+------------+---------------------+---------------+
```
