# WordPress

## Description

Commandes utilisables avec la recette WordPress

## Commands

- **[auto-login](auto-login)** - Accès admin en un clic
- **[update-appconfig](update-appconfig)** - Met à jour la configuration pour l'utilisation avec `dvs`
- **[debug-mode](debug-mode)** - Active ou désactive le debug WordPress
- **[wp](wp)** - Raccourci vers la commande `wp-cli`
- **[user-list](user-list)** - Liste les accès utilisateur
- **[user-password](user-password)** - Modifie le mot de passe d'un compte utilisateur
- **[maintenance](maintenance)** - Active ou désactive le mode maintenance
