# Recettes

## Description

Les recettes sont des images docker prêtes à être utilisées et contenant un ensemble de fonctionnalités adaptées à l'application cible. L'image d'une recette peut être basée sur une autre recette et bénéficie automatiquement des fonctionnées associées. Par exemple la recette `Prestashop` est construite sur la base de la recette `LAMP`, toutes les commandes `LAMP` sont donc accessibles dans la recette `Prestashop`.
La documentation d'une recette liste uniquement les commandes spécifique, se reporter à la recette de base pour plus d'infos sur les commandes additionnelles.

## Recettes

- **[lamp](lamp)** - Application générique LAMP
- **[lamp-framework](lamp-framework)** - Application LAMP basée sur un framework
- **[prestashop](prestashop)** - Application Prestashop
- **[wordpress](wordpress)** - Application WordPress
