# debug-mode

Active ou désactive le mode debug de prestashop

## Usage

```bash
dvs debug-mode [action]
```

## Argument

- `action` - commande à choisir parmi:
    - `status`: Affiche le status actif/inactif
    - `toggle`: Bascule le statut
    - `enable`: Active le mode debug de prestashop
    - `disable`: Désactive le mode debug de prestashop
