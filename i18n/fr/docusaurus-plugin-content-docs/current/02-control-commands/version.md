# dvs version

Afficher les informations de version.

## Description

La commande `version` affiche la version actuelle du binaire dvsctl et des composants associés.

## Utilisation

```bash
dvs version
```

## Ce qu'elle affiche

- Version actuelle de dvsctl
- ID d'installation (si configuré)
- Nom de l'organisation (si configuré)

## Exemples

```bash
# Afficher la version
dvs version
```

## Exemple de sortie

```
dvsctl version: 0.3.17
Installation ID: 42
Organisation: Mon Entreprise
```
