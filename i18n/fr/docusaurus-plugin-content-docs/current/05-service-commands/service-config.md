# dvs service config

Configure un service en éditant son fichier de configuration.

## Usage

```bash
dvs service config [service]
```

La commande ouvre un éditeur (nano) pour modifier le fichier de configuration du service. Si le fichier de configuration est modifié, le service est automatiquement redémarré.

Voir la liste des services disponibles sur [la page des services](/docs/service-commands/).
