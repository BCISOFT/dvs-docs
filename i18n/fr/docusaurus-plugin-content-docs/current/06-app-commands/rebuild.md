# dvs rebuild

Reconstruit l'image docker d'une application.

## Usage

```bash
dvs rebuild [name|all]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `name` | Nom de l'application Ã  reconstruire (optionnel) |
| `all` | Reconstruit toutes les applications |

Si aucun argument n'est fourni, reconstruit l'application courante.
