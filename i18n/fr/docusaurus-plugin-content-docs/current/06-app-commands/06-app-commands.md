# Application

## Description

Gestion des applications.

## Commandes

- **[init](init)** - Initialisation une application dans le dossier courant
- **[create](create)** - Crée et installe une nouvelle application dans le dossier courant
- **[start](start)** - Démarre une application
- **[stop](stop)** - Stoppe une application
- **[list](list)** - Liste les applications installées
- **[rebuild](rebuild)** - Reconstruit l'image docker d'une application
- **[destroy](destroy)** - Détruit l'image docker d'une application
- **[app-folder](app-folder)** - Renvoi le chemin d'une application
- **[shell](shell)** - Ouvre un shell dans le conteneur de l'application
- **[dump-config](dump-config)** - Exporte la configuration d'une application
- **[logs](logs)** - Affiche les logs de l'application
