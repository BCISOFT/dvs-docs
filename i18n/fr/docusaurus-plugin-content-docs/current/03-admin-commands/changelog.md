# dvs changelog

Affichage la liste des modifications par version.

## Description

La commande `changelog` permet d'afficher la liste des versions avec le détail des modifications et nouvelles fonctionnalités:

- affiche la liste complète des versions si la version actuelle est à jour
- affiche uniquement les nouveautés avec la prochaine mise à jour

## Utilisation

```bash
dvs changelog
```
